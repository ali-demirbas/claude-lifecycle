claude-lifecycle

Most AI marketing tools will happily write you a 12-step journey from zero data, benchmarks included, all of it made up. This plugin works the other way: it scores how good your data actually is first, and only designs the journeys that data can support.

Below is a real run against a fictional edtech dataset, 40 users over 60 days. The engine found that every user who finished their first lesson went on to subscribe, and none of the ones who skipped it did, so the whole trial journey is built around getting that one lesson done rather than around selling anything. That kind of finding is what drives the output, not a template.

  • Journey canvas: five journeys as flow maps you can click through, from entry rules to exits
  • Message copy: the actual email, push and in-app text for each step, with A/B variants and character counts checked in code, not by eye
